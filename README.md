# MIT 6.824 学习笔记

这是李响的 MIT 6.824 课程学习笔记。

## 课程简介

MIT 6.824 是麻省理工学院开设的一门关于分布式系统的课程，涵盖了分布式系统的基本概念、设计和实现。课程内容包括分布式系统的容错、复制、一致性、分布式存储等。
